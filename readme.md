# Cài đặt dự án

Lệnh tạo file package.json
> npm init

Lệnh tạo file bower.json 
> bower init

Cài một plugins nào đó cho node đánh lệnh 
> npm i gulp

Cài một plugins nào đó cho node mà chạy global đánh lệnh 
> npm i -g gulp 

Cài đúng version nào đó đánh 
> npm i gulp@3.4.5

Tương tự cho bower
